## 1. 백테스트란?

### 🎯 한 줄 요약

**"과거 데이터로 투자 전략을 미리 연습해보는 것"**

### 📖 자세한 설명

백테스트는 마치 "타임머신 시뮬레이션"과 같아요!

> 🎮 **비유**: 게임을 하기 전에 튜토리얼을 하는 것처럼,
> 실제 돈을 투자하기 전에 가상의 돈으로 미리 테스트해보는 거에요.

**예시:**

- 2020년~2023년까지의 비트코인 가격 데이터를 가져옴
- "가격이 5% 떨어지면 사고, 10% 오르면 팔자"라는 전략을 적용
- 이 전략으로 3년간 얼마를 벌었는지 확인!

### ⚠️ 주의사항

- 백테스트 결과가 좋다고 실제로 수익이 보장되는 건 아니에요
- 과거 성과 ≠ 미래 성과 (하지만 참고는 됩니다!)

## 2. 핵심 용어 쉬운 설명

### 💰 초기 자금 (Initial Balance)

```
🎯 한 줄: 백테스트를 시작할 때 가상으로 가지고 있는 돈

📝 예시: 초기 자금 $10,000 = "1만 달러로 시작해서 테스트"
```

### 📅 테스트 기간 (Date Range)

```
🎯 한 줄: 테스트에 사용할 과거 기간

📝 예시: 2022-01-01 ~ 2023-12-31 = "2년간의 데이터로 테스트"

💡 팁: 최소 1년 이상의 기간을 권장해요!
     (상승장, 하락장 모두 포함되어야 신뢰도 UP)
```

### 🎚️ 레버리지 (Leverage)

```
🎯 한 줄: 내 돈을 빌려서 더 크게 거래하는 것

📊 쉬운 비유:
   - 1x = 내 돈 그대로 (안전)
   - 3x = 내 돈의 3배로 거래 (조금 위험)
   - 10x = 내 돈의 10배로 거래 (매우 위험!)

📝 예시: 
   $100있는데 10x 레버리지 = $1,000처럼 거래
   → 10% 오르면 100% 이익 (100 → 200)
   → 10% 내리면 100% 손실 (100 → 0) 💀

⚠️ 초보자 권장: 1x~3x만 사용하세요!
```

### 🛡️ 마진 모드 (Margin Mode)

```
🎯 한 줄: 손실을 어디까지 감당할지 정하는 것

🔵 격리 마진 (Isolated) - 안전벨트 O
   → 이 거래에 넣은 돈만 잃을 수 있음
   → "최대 손실 = 투자한 금액"
   
🔴 교차 마진 (Cross) - 안전벨트 X
   → 계좌 전체가 담보가 됨
   → "최대 손실 = 계좌 전체"

⚠️ 초보자: 반드시 '격리' 선택!
```

### 📈 승률 (Win Rate)

```
🎯 한 줄: 전체 거래 중 이긴 비율

📝 예시: 100번 거래 → 60번 이김 = 승률 60%

⚠️ 주의: 승률만 높다고 좋은 게 아니에요!
   - 승률 90%인데 이길 때 $10, 질 때 $100이면?
   - → 결국 손해! (-$$)
```

### 📉 최대 손실 (MDD, Maximum Drawdown)

```
🎯 한 줄: 테스트 기간 중 돈이 가장 많이 줄어든 순간

📊 쉬운 비유: 롤러코스터의 가장 무서운 하락 구간

📝 예시:
   MDD 20% = $100 → 최저 $80까지 떨어졌음
   MDD 50% = $100 → 최저 $50까지 떨어졌음 😱

💡 기준:
   ✅ MDD 10% 이하: 매우 안정적
   ⚡ MDD 20% 이하: 적절한 위험
   ⚠️ MDD 30% 이상: 심장이 쫄깃함
   ❌ MDD 50% 이상: 위험한 전략!
```

### ⚖️ Profit Factor (수익 비율)

```
🎯 한 줄: 총 이익 ÷ 총 손실

📝 예시:
   이긴 거래 합계: $200
   진 거래 합계: $100
   → Profit Factor = 200 ÷ 100 = 2.0

💡 해석:
   PF < 1.0: 손실 중 💸
   PF 1.0~1.5: 별로
   PF 1.5~2.0: 괜찮음 👍
   PF 2.0~3.0: 좋음 ⭐
   PF 3.0+: 매우 좋음 🌟
```

### 📐 샤프 비율 (Sharpe Ratio)

```
🎯 한 줄: "얼마나 안정적으로 돈을 벌었나?"를 점수로 표현

📊 쉬운 비유: 
   - 샤프 높음 = 편안한 드라이브
   - 샤프 낮음 = 롤러코스터 드라이브

💡 해석:
   < 0: 손실 + 불안정 ❌
   0~1: 보통
   1~2: 괜찮음
   2~3: 좋음 ⭐
   > 3: 매우 좋음 🌟
```

## 4. 결과 해석 방법

### 📊 결과 점수표 이해하기

백테스트 결과에는 **전략 점수표**가 표시됩니다:

```
📊 전략 평가 점수표     [A등급]

🛡️ 안정성:     ⭐⭐⭐⭐⭐  (매우 안정적!)
💰 수익성:     ⭐⭐⭐⭐☆   (높은 수익!)
⚠️ 위험도:     ⭐⭐☆☆☆   (낮은 위험!)
✅ 신뢰도:     ⭐⭐⭐⭐☆   (높은 신뢰!)
```

### 🏆 등급 해석

| 등급 | 의미 | 권장 |
|------|------|------|
| **S** | 최고의 전략 | 실전 적용 적극 고려 |
| **A** | 좋은 전략 | 실전 적용 권장 |
| **B** | 보통 전략 | 조금 더 개선 필요 |
| **C** | 아쉬운 전략 | 다른 설정 시도 |
| **D** | 위험한 전략 | 사용 비권장 |

### 📈 좋은 결과의 기준

```
✅ 좋은 전략 기준:
   - 총 수익률: 양수 (+%)
   - 승률: 45% 이상  
   - MDD: 30% 이하
   - Profit Factor: 1.5 이상
   - 샤프 비율: 1.0 이상
```

## 5. 초보자 꿀팁 10가지

### 🎯 처음엔 레버리지 1x로
>
> 레버리지는 "양날의 검"이에요. 이익도 크지만 손실도 커져요.
> 처음엔 1x로 시작해서 전략을 검증하세요.

### 📅 최소 1년 이상 데이터로
>
> 너무 짧은 기간은 신뢰도가 낮아요.
> 상승장과 하락장 모두 포함된 기간을 선택하세요.

### ⚖️ 승률보다 Profit Factor를
>
> 승률 90%여도 손해볼 수 있어요.
> Profit Factor 1.5 이상인지 먼저 확인하세요.

### 📉 MDD 30% 이상이면 조심
>
> MDD가 높으면 실제로 투자할 때 멘탈이 무너질 수 있어요.
> MDD 20% 이하가 편안해요.

### 🔄 다양한 기간에서 테스트
>
> 한 기간에서만 좋으면 "운"일 수 있어요.
> 2~3개 다른 기간에서도 테스트해보세요.

### 🛡️ 항상 격리 마진 선택
>
> 교차 마진은 계좌 전체를 잃을 수 있어요.
> 안전하게 격리 마진만 사용하세요.

### 💸 펀딩 피 적용 권장
>
> 선물 거래는 펀딩 피가 수익을 갉아먹어요.
> 더 현실적인 결과를 위해 펀딩 피를 적용하세요.

### 📊 여러 전략 비교하기
>
> 하나의 전략만 보지 마세요.
> "백테스트 비교" 탭에서 여러 전략을 비교해보세요.

### 🎓 천천히 배우기
>
> 처음부터 모든 걸 이해할 필요 없어요.
> 하나씩 천천히 익혀가세요.

### ⚠️ 과적합 주의
>
> 과거에만 잘 맞는 전략은 미래에 안 통해요.
> "너무 완벽한" 결과는 의심하세요.

## 6. 자주 묻는 질문 FAQ

### Q1. 백테스트 결과가 좋으면 실제로도 수익날까요?
>
> **A**: 100% 보장은 아니에요! 하지만 백테스트로 검증된 전략이
> 검증 안 된 전략보다는 훨씬 신뢰할 수 있어요.
>
> ⚠️ 백테스트 결과 ≠ 실제 결과 (참고용!)

### Q2. 어떤 기간으로 테스트해야 하나요?
>
> **A**: 최소 1년, 권장 2~3년!
>
> - 상승장만 포함된 기간 → 신뢰도 낮음
> - 상승장 + 하락장 + 횡보장 → 신뢰도 높음

### Q3. 레버리지는 몇 배가 적당한가요?
>
> **A**: 초보자는 **1x~3x** 권장!
>
> - 1x: 매우 안전 (원금 손실 제한)
> - 3x: 적당한 위험
> - 10x+: 고수용 (청산 위험 높음)

### Q4. 승률이 높으면 좋은 전략인가요?
>
> **A**: 아니에요! **승률 vs 손익비**를 함께 봐야 해요.
>
> - 승률 90% + 이길 때 $1, 질 때 $100 = 손실
> - 승률 40% + 이길 때 $100, 질 때 $10 = 수익!
>
> → **Profit Factor**로 판단하세요.

### Q5. MDD가 뭔가요? 왜 중요한가요?
>
> **A**: Maximum Drawdown = 최대 손실폭
>
> - MDD 50%면 $100 → $50까지 떨어질 수 있다는 뜻
> - 실제로 투자할 때 이 정도 손실을 견딜 수 있는지 생각해보세요
>
> → **MDD 30% 이하**를 권장!

### Q6. 샤프 비율은 뭔가요?
>
> **A**: "안정성 점수"라고 생각하세요!
>
> - 수익률이 같아도 오르락내리락이 심하면 샤프 낮음
> - 꾸준히 오르면 샤프 높음
>
> → **2.0 이상**이면 좋은 전략!

### Q7. 펀딩 피는 적용해야 하나요?
>
> **A**: **적용 권장!** (더 현실적인 결과)
>
> - 적용 안 하면 실제보다 좋게 나옴
> - 선물 거래에는 펀딩 피가 있으니까요

### Q8. 거래 수가 적어도 괜찮나요?
>
> **A**: 최소 30번 이상 권장!
>
> - 10번 거래 → 통계적으로 신뢰할 수 없음
> - 100번 이상 → 신뢰도 높음

### Q9. 백테스트 말고 실제로 테스트해보고 싶어요
>
> **A**: **페이퍼 트레이딩**을 해보세요!
>
> - 실제 시장에서 가상의 돈으로 테스트
> - 백테스트보다 더 현실적

### Q10. 어떤 전략이 가장 좋은가요?
>
> **A**: "최고의 전략"은 없어요!
>
> - 시장 상황에 따라 잘 맞는 전략이 달라요
> - 본인의 투자 성향에 맞는 전략을 선택하세요
> - 위험을 감수할 수 있다면 → 공격적 전략
> - 안정적인 수익을 원한다면 → 보수적 전략
