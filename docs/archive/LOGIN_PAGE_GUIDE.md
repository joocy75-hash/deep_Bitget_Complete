# 🎨 새로운 로그인 페이지 가이드

## 📋 개요

UIPro 스타일을 적용한 완전히 새로운 디자인의 모던 로그인 메인페이지입니다.

## 🚀 접속 방법

- **새 로그인 페이지**: http://localhost:3001/login
- **기존 로그인 페이지** (백업): http://localhost:3001/login-old

## ✨ 주요 기능

### 1. 🎯 모던한 UI/UX 디자인

#### 시각적 요소
- **그라디언트 배경**: 퍼플-핑크 그라디언트 애니메이션 배경
- **글래스모피즘**: 반투명 효과와 블러 처리
- **플로팅 애니메이션**: 부드럽게 움직이는 그라디언트 블롭 3개
- **페이드인 효과**: 페이지 로드 시 0.6초 페이드인 애니메이션

#### 컬러 팔레트
- Primary: `#667eea` → `#764ba2` (그라디언트)
- Success: `#52c41a` (강한 비밀번호)
- Warning: `#faad14` (보통 비밀번호)
- Danger: `#f5222d` (약한 비밀번호)

### 2. 📱 완벽한 반응형 디자인

#### 화면 크기별 최적화
- **데스크톱** (1024px 이상)
  - 2컬럼 레이아웃 (브랜드 섹션 + 폼 섹션)
  - 왼쪽: 브랜드 소개 및 통계
  - 오른쪽: 로그인/회원가입 폼

- **태블릿** (768px ~ 1024px)
  - 1컬럼 세로 레이아웃
  - 브랜드 섹션 상단
  - 폼 섹션 하단

- **모바일** (768px 이하)
  - 모바일 최적화 레이아웃
  - 폰트 크기 및 간격 조정
  - 터치 친화적 버튼 크기

### 3. 🔐 인증 기능

#### 로그인
- 이메일/비밀번호 입력
- 로그인 상태 유지 옵션
- 비밀번호 찾기 링크
- 소셜 로그인 (Google, Kakao)
- 테스트 계정 정보 표시

#### 회원가입
- 이메일 (유효성 검사)
- 이름 (최소 2자)
- 전화번호 (숫자와 하이픈)
- 비밀번호 (최소 8자)
- **비밀번호 강도 표시기** ⭐ NEW!
  - 실시간 강도 분석
  - 진행 바와 텍스트 표시
  - 색상 코딩 (빨강/주황/초록)
- 비밀번호 확인 (일치 여부 검사)

#### 2단계 인증 (2FA)
- 6자리 TOTP 코드 입력
- 대형 숫자 입력 필드
- 자동 제출 (6자리 입력 시)

## 🚦 상태 관리

### 로컬 상태
- `activeView`: 현재 뷰 ('login' | 'register' | '2fa')
- `error`: 에러 메시지
- `loading`: 로딩 상태
- `requires2FA`: 2FA 필요 여부
- `tempCredentials`: 임시 인증 정보
- `totpCode`: TOTP 코드
- `rememberMe`: 로그인 유지
- `passwordStrength`: 비밀번호 강도 (0-100)
- `isPageLoaded`: 페이지 로드 상태

### Context API
- `AuthContext`: 인증 상태 관리
  - `login()`: 로그인 함수
  - `isAuthenticated`: 인증 여부
  - `loading`: 로딩 상태

## 🔄 API 연동

### 엔드포인트

#### 로그인
```
POST /auth/login
Body: { email, password, totp_code? }
Response: { access_token, requires_2fa? }
```

#### 회원가입
```
POST /auth/register
Body: { email, password, passwordConfirm, name, phone }
Response: { access_token, user }
```

#### Google OAuth
```
GET /auth/google/login
Redirect: Google OAuth 페이지
```

#### Kakao OAuth
```
GET /auth/kakao/login
Redirect: Kakao OAuth 페이지
```

## 🎬 사용자 플로우

### 로그인 플로우
1. 이메일/비밀번호 입력
2. "로그인" 버튼 클릭
3. (2FA 활성화 시) 6자리 코드 입력
4. 인증 성공 → 대시보드로 이동

### 회원가입 플로우
1. "회원가입" 탭 선택
2. 정보 입력 (이메일, 이름, 전화번호, 비밀번호)
3. 비밀번호 강도 확인
4. "가입하기" 버튼 클릭
5. 성공 → "로그인" 탭으로 자동 전환

### 소셜 로그인 플로우
1. "Google로 계속하기" 또는 "카카오로 계속하기" 클릭
2. OAuth 페이지로 리다이렉트
3. 인증 완료 → 콜백 URL로 리턴
4. 토큰 저장 → 대시보드로 이동

## 🐛 문제 해결

### 스타일이 적용되지 않음
- `LoginNew.css` 파일이 올바르게 import 되었는지 확인
- 브라우저 캐시 클리어
- 개발 서버 재시작

### 2FA가 작동하지 않음
- 백엔드 서버가 실행 중인지 확인
- TOTP 앱 시간 동기화 확인

### 소셜 로그인 실패
- 환경 변수 `VITE_API_URL` 확인
- OAuth 클라이언트 ID/Secret 확인
- 리다이렉트 URI 설정 확인
