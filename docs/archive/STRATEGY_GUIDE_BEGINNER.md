## 🤔 전략이란 무엇인가?

**트레이딩 전략**은 "언제 사고, 언제 팔지"를 정해놓은 규칙입니다.

### 왜 전략이 필요한가?

| 감정 트레이딩 ❌ | 전략 트레이딩 ✅ |
|-----------------|-----------------|
| 오를 것 같아서 매수 | RSI가 30 이하면 매수 |
| 무서워서 손절 안 함 | -2% 도달 시 자동 손절 |
| 욕심 부려서 익절 못함 | +4% 도달 시 자동 익절 |
| 일관성 없음 | 항상 동일한 규칙 적용 |

## 🚀 간단 전략 생성기 사용법

### Step 1: 전략 유형 선택

메뉴에서 **"전략 관리"** → **"🌟 간단 전략 만들기"** 탭을 클릭합니다.

4가지 전략 중 하나를 선택하세요:

```
✨ 골든크로스 전략 (추천 - 초보자용)
🔄 RSI 반전 전략 (쉬움)
📈 추세추종 전략 (보통)
🚀 돌파 전략 (보통)
```

### Step 2: 조건 설정

#### 거래 코인 선택

- **BTCUSDT** - 비트코인 (가장 안정적)
- **ETHUSDT** - 이더리움 (두 번째 추천)
- **기타** - XRP, SOL 등

#### 시간봉 선택

| 시간봉 | 특징 | 대상 |
|--------|------|------|
| 15분봉 | 빠른 거래, 변동성 큼 | 숙련자 |
| **1시간봉** | 균형잡힌 거래 | **초보자 추천** ⭐ |
| 4시간봉 | 느린 거래, 안정적 | 바쁜 직장인 |
| 일봉 | 장기 투자 | 스윙 트레이더 |

### Step 3: 리스크 관리 설정

**가장 중요한 단계입니다!**

```
손절 (Stop Loss): 2-3% 추천
익절 (Take Profit): 4-6% 추천
손익비: 최소 1:2 이상
```

### Step 4: 전략 저장

이름을 입력하고 "전략 생성하기" 버튼을 클릭합니다.

## 📊 전략 유형별 설명

### ✨ 골든크로스 전략 (가장 쉬움)

**원리**: 단기 이동평균선이 장기 이동평균선을 위로 뚫으면 상승 추세 시작

```
매수 조건: 9일 EMA가 21일 EMA를 상향 돌파
매도 조건: 9일 EMA가 21일 EMA를 하향 돌파
```

**장점**:

- 이해하기 쉬움
- 큰 추세를 잘 잡음
- 거짓 신호가 적음

**단점**:

- 신호가 늦을 수 있음
- 횡보장에서 손실 가능

---

### 🔄 RSI 반전 전략

**원리**: 가격이 너무 떨어지면(과매도) 반등할 가능성이 높음

```
매수 조건: RSI가 30 이하일 때 (과매도)
매도 조건: RSI가 70 이상일 때 (과매수)
```

**장점**:

- 저점에서 매수 가능
- 명확한 진입 기준

**단점**:

- 강한 하락장에서는 계속 떨어질 수 있음
- 추세장보다 횡보장에 유리

---

### 📈 추세추종 전략

**원리**: 추세는 지속되는 경향이 있다

```
매수 조건: 강한 상승 추세 확인
매도 조건: 추세 전환 신호
```

**장점**:

- 큰 수익 가능
- 추세장에 강함

**단점**:

- 추세 시작을 놓칠 수 있음
- 추세 전환 시 손실

---

### 🚀 돌파 전략

**원리**: 저항선을 뚫으면 급등하는 경우가 많음

```
매수 조건: 최근 고점 돌파
매도 조건: 목표가 도달 또는 손절
```

**장점**:

- 폭발적 수익 가능
- 명확한 진입점

**단점**:

- 거짓 돌파 위험
- 빈도가 낮음

## 📈 지표 설명

### RSI (상대강도지수)

> **0~100 사이의 값으로 과매수/과매도를 측정**

| RSI 값 | 의미 | 행동 |
|--------|------|------|
| 0-30 | 과매도 (많이 떨어짐) | 매수 고려 🟢 |
| 30-70 | 중립 | 관망 |
| 70-100 | 과매수 (많이 올랐음) | 매도 고려 🔴 |

**설정 팁**:

- RSI 기간: 14 (기본값, 가장 널리 사용)
- 매수 기준: 30 이하
- 매도 기준: 70 이상

---

### EMA (지수이동평균선)

> **최근 가격에 더 비중을 두는 평균선**

```
단기 EMA (9, 12): 빠르게 반응
장기 EMA (21, 26, 50): 느리게 반응
```

**골든크로스**: 단기선이 장기선 위로 → 매수 신호 📈
**데드크로스**: 단기선이 장기선 아래로 → 매도 신호 📉

**추천 설정**:

- 단기 EMA: 9 또는 12
- 장기 EMA: 21 또는 26

## 🛡️ 리스크 관리

### 손절 (Stop Loss) - 가장 중요

> **얼마나 손해보면 포기할 것인가**

```
초보자 추천: 2-3%
```

**예시**:

- 100만원 투자, 손절 2%
- 98만원이 되면 자동 매도
- 최대 손실: 2만원으로 제한

### 익절 (Take Profit)

> **얼마나 이익나면 수익 확정할 것인가**

```
초보자 추천: 손절의 2배 (손절 2% → 익절 4%)
```

### 손익비 (Risk:Reward)

> **손실 대비 이익 비율**

```
손익비 = 익절 / 손절

예: 손절 2%, 익절 4% → 손익비 1:2
```

| 손익비 | 평가 |
|--------|------|
| 1:1 이하 | ❌ 위험 |
| 1:1.5 | 보통 |
| **1:2 이상** | ✅ 좋음 |
| 1:3 이상 | 🌟 우수 |

### 포지션 크기

> **한 번에 얼마나 투자할 것인가**

```
초보자 추천: 전체 자금의 10-20%
```

**왜 100%를 넣으면 안 되나?**

- 한 번의 실패로 회복 불가
- 여러 기회에 분산 투자 불가

### 레버리지

> **빌린 돈으로 투자 규모 확대**

```
1x: 레버리지 없음 (가장 안전)
5x: 수익/손실 5배
10x: 수익/손실 10배 (위험)
```

| 레버리지 | 위험도 | 추천 대상 |
|----------|--------|----------|
| **1x** | 낮음 | **초보자 필수** ⭐ |
| 2-3x | 보통 | 경험자 |
| 5x 이상 | 높음 | 전문가 |

### Q: 손절을 안 하면 안 되나요?

**A**: 절대 안 됩니다! 손절 없이는 한 번의 실패로 전재산을 잃을 수 있습니다.

**A**: 네, 하지만 그만큼 많이 잃습니다. 초보자는 반드시 1x로 시작하세요.
